# 🥋 태권커리어 (Taekwon Career)

태권도장과 사범을 연결하는 대한민국 대표 **태권도 전문 구인구직 플랫폼**입니다.  
기존의 SNS, 단체 채팅방, 커뮤니티 분산형 구인구직 방식에서 벗어나, 국세청 사업자 진위여부 API를 통한 신뢰성 높은 도장 정보와 체계적인 온라인 이력서 관리 및 지원 프로세스를 지원하는 디지털 매칭 서비스입니다.

👉 **서비스 주소**: [https://taekwoncareer.co.kr](https://taekwoncareer.co.kr) (기존: [https://taekwondo-2026-kh.web.app](https://taekwondo-2026-kh.web.app))

---

## 🌟 주요 제공 기능

### 1. 사용자 서비스 (Customer Portal)
* **채용공고 관리**: 관장님은 빠르고 편리하게 공고 정보를 기입하고 등록할 수 있으며, 공고 상단 고정 노출(프리미엄 광고)을 유료 신청할 수 있습니다.
* **이력서 및 즉시 지원**: 사범님은 경력과 자격증 등 자기소개 정보를 온라인 이력서로 등록하고, 마음에 드는 채용공고에 즉시 지원할 수 있습니다.
* **1:1 고객 문의**: 서비스 이용에 필요한 문의 유형(일반 문의/이용 방법/결제 문의/신고 등)을 선택하여 이메일 답변용 1:1 문의를 등록할 수 있습니다. (전화번호 수집 없이 개인정보 동의 팝업 탑재)
* **커뮤니티 공간**: 사범 및 관장 회원들이 자유롭게 글을 쓰고 실시간 댓글을 통해 네트워킹할 수 있는 공간을 제공합니다. (로컬 데이터 연동 완료)

### 2. 관리자 페이지 (Backoffice Admin)
* **대시보드 통계**: 가입 회원 수, 전체 등록 공고 수, 이력서 등록 및 지원 현황 통계를 차트와 집계 수치로 제공합니다.
* **지원 및 이력서 관리**: 들어온 지원서의 상태를 검토하여 합격/불합격/면접제안 등으로 업데이트하면 **EmailJS를 통해 사범님께 실시간 합격 알림 이메일이 자동 발송**됩니다.
* **1:1 문의 처리**: 실시간으로 연동된 문의 데이터를 조회하고, 상세 모달창을 통해 즉각 피드백 답변을 작성하여 업데이트할 수 있습니다.
* **동적 카운터 알림 뱃지**: 신규 문의 접수, 신규 지원서 제출, 최근 3일 이내 신규 이력서 등록에 따라 사이드바 메뉴 옆에 실시간 붉은색 알림 뱃지가 갱신되어 관리 편의를 돕습니다.

## 🛠️ 기술 스택 (Tech Stack)

### Frontend
* **Core**: `HTML5` / `JavaScript (ES6+)`
* **Styling**: `Vanilla CSS` (모던 그리드/플렉스 레이아웃, 미려한 다크 모드 지원, 컴포넌트 뱃지/모달 팝업 가이드 적용)
* **Architecture**: 단일 소스 리소스 제어로 빠른 속도를 자랑하는 반응형 **Single Page Application (SPA)** 및 커스텀 라우팅 시스템

### Database & Serverless
* **Authentication**: `Firebase Auth` (이메일 가입 및 회원별 전용 대시보드 접근 격리)
* **Database**: `Cloud Firestore` (실시간 컬렉션 동기화, 사용자별 권한 필터 및 보안 규칙 탑재)
* **Hosting**: `Firebase Hosting` (캐싱 방지 무효화 헤더 및 SPA redirection 연동)

### APIs & Libraries
* **국세청 API**: 관장 회원 가입 시 사업자 정보(사업자등록번호, 대표자명, 개업일자)의 실시간 정상 여부 조회 연동
* **EmailJS SDK**: 지원서 상태 승인/변경 시 자동 알림 메일링 인프라 구축

## 📂 프로젝트 구조

```text
├── index.html          # 사용자 메인 및 SPA 레이아웃 통합 뷰 포털
├── index.css           # 전역 스타일 및 사용자 페이지별 테마 CSS
├── index.js            # Firebase 연동, 폼 제출 및 클라이언트 SPA 라우팅 로직
├── admin.html          # 관리자 대시보드 및 백오피스 통합 관리 뷰
├── admin.css           # 백오피스 전용 모던 대시보드 컴포넌트 CSS
├── admin.js            # 실시간 컬렉션 Fetch, 답변 등록, 지원 상태 및 알림 뱃지 갱신 로직
├── firestore.rules     # 컬렉션별(Users, Jobs, Resumes, Apply, Inquiries) 보안 규칙
├── firebase.json       # 호스팅 Redirects, Headers(Cache-Control) 구성 파일
└── README.md           # 프로젝트 가이드라인 문서
```
